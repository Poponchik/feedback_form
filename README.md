Run application by following steps:

1. Create new postgreSQL database called test_task

2. Install client dependencies and run frontend
 2.1. cd client
 2.2 npm i
 2.3 npm start

3. Copy .env.dist to .env, install server dependencies and run backend
 3.1 cd server
 3.2. cp .env.dist .env
 3.3 npm i
 3.4 npm run dev
